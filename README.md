# Cook

程序员做饭指南 - 使用 VitePress 构建的中文菜谱网站

## 项目介绍

这是一个面向程序员的烹饪指南，用代码的视角来看待烹饪，让烹饪变得简单有趣。

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览生产版本
npm run docs:preview
```
